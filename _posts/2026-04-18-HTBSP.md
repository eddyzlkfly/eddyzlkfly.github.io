## Challenge Scenario

All the coolest ghosts in town are going to a Haunted Houseparty - can you prove you deserve to get in?

## Description

In this challenge, we are given a binary file and required to find the correct password to gain access.

## Initial Execution

First, we execute the provided binary:


Instead of guessing the password, we inspect the binary directly.

