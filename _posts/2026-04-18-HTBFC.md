## Challenge Scenario

All the coolest ghosts in town are going to a Haunted Houseparty - can you prove you deserve to get in?

## Initial Execution

To begin, we enter:

```
start
```

## Discovering Hidden Commands

There is no need to follow all the options provided by the application, as the solution can be achieved by directly using the hidden command discovered in the client-side code.

After interacting with the application, we inspect the browser using **Developer Tools**

### Step

- Open **DevTools**
- Go to the **Network** tab
- Look for relevant files → `option.js`

## Result

Inside `option.js`, we find a list of all possible commands:

![/assets/img/imgHTBFC(1).png](/assets/img/imgHTBFC(1).png)

Aside from the normal options, there is also a hidden field.

### This indicates that:

- There is a **hidden command**
- It is not shown in the UI
- It may be required to progress or solve the challenge

## Exploitation

After discovering the hidden secret command from `option.js`, we input it directly into the application:

```bash
Blip-blop, in a pickle with a hiccup! Shniggity-shmack
```
